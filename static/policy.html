Общие положения.

Настоящая политика конфиденциальности (далее — «Политика») определяет порядок обработки персональных данных пользователей сервиса StreamYam (далее — «Сервис»).

Сервис обеспечивает конфиденциальность персональных данных, которые предоставляет пользователь, и принимает необходимые меры для их защиты от несанкционированного доступа третьих лиц.
При регистрации на Сервисе пользователь предоставляет свои персональные данные, включая электронную почту и имя аккаунта. Эти данные используются для предоставления персонализированного доступа и управления настройками виджета. Сервис не передаёт персональные данные третьим лицам без согласия пользователя, за исключением случаев, предусмотренных законодательством РФ.
Для обеспечения безопасности персональных данных Сервис использует современные методы шифрования и аутентификации. Доступ к персональным данным имеют только уполномоченные сотрудники Сервиса, которые обязуются не разглашать эти данные третьим лицам.
Пользователь имеет право на доступ к своим персональным данным, их исправление, удаление, ограничение обработки, перенос данных, возражение против обработки данных и не подтверждение действиям автоматизированных систем принятия решений. Чтобы воспользоваться этими правами, необходимо связаться с нами по адресу электронной почты support@streamyam.ru.
Мы оставляем за собой право изменять условия Политики без предварительного уведомления. Новая версия Политики вступает в силу с момента её публикации на странице https://streamyam.ru/. Продолжение использования Сервиса после внесения таких изменений означает принятие вами этих изменений.
